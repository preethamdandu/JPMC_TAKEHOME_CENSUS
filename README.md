# Census Income Classification & Segmentation

Predict whether a person earns >$50K (Objective 1) and segment the population for marketing (Objective 2), using 40 demographic/employment variables from U.S. Census data.

- **Objective 1:** Random Forest classifier — best precision and F1 for targeted marketing
- **Objective 2:** K-Prototypes (k=6) segmentation — strongest income separation across segments

See `report.pdf` for the full client report (7 pages).

## Prerequisites

- Python 3.8+
- pip

## Setup

```bash
pip install -r requirements.txt
```

### Data

Place the census data file in `data/`:

```
data/census-bureau.data       # ~199K rows, comma-delimited, no header
data/census-bureau.columns    # already included — column names, one per line
```

The data file (`censusbureau.data` or `census-bureau.data`) is provided separately with the project. If your file is named `censusbureau.data`, either rename it or pass `--data censusbureau.data` to the scripts.

## How to Run

Run all commands from the **repo root**.

### 1. Explore the data

```bash
python src/data_loader.py          # loads data, prints shape and target balance
python src/01_explore_data.py      # prints class imbalance, numeric stats, missing values
```

### 2. Train the classification model (Objective 1)

```bash
python src/train_classifier.py             # with grid search (~5-10 min)
python src/train_classifier.py --no-grid   # without grid search (~1 min)
```

Saves to `outputs/`:
- `classifier_model.pkl` — full pipeline (preprocessing + Random Forest)
- `classifier_metrics.txt` — test set metrics, classification report, confusion matrix

### 3. Train the segmentation model (Objective 2)

```bash
python src/train_segmentation.py           # K-Prototypes k=6 on full data (~10-15 min)
python src/train_segmentation.py --sample 30000   # faster on a sample
```

Saves to `outputs/`:
- `segmentation_model.pkl` — K-Prototypes model + scaler + column info
- `segment_assignments.csv` — segment label and income label per row

### 4. Generate report figures (optional)

```bash
python generate_report_figures.py
```

Generates 6 plots in `outputs/plots/` (class distribution, feature importance, confusion matrix, ROC curve, PR curve, segment rates). Requires `classifier_model.pkl` and `segment_assignments.csv` to exist first.
